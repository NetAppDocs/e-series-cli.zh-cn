= 停止驱动器定位
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
使用`stop drive locate`命令可关闭由`sStart drive locate`命令打开的驱动器上的指示灯。



== 支持的阵列

此命令将适用场景 任意单个存储阵列、包括E2700、E5600、E2800、E5700、 EF600和EF300阵列、前提是安装了所有SMcli软件包。



== 角色

要在E2800、E5700、EF600或EF300存储阵列上执行此命令、您必须具有存储管理员角色。



== 语法

[listing]
----
stop drive locate
----


== Parameters

无



== 最低固件级别

6.10
